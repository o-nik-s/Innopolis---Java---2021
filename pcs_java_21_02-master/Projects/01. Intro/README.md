## Количество информации

* Бит - минимальная единица информации - {0, 1}

```
0 + 0 = 0
1 + 0 = 1
0 + 1 = 1
1 + 1 = 10 = 2(10)
1 + 1 + 1 = 11 = 3(10)

N = 1
0 -> 0(10)
1 -> 1(10)
count = 2, min = 0, max = 1

N = 2
00 -> 0(10)
01 -> 1(10)
10 -> 2(10)
11 -> 3(10)
count = 4, min = 0, max = 3

N = 3
000 -> 0(10)
001 -> 1(10)
010 -> 2(10)
011 -> 3(10)
100 -> 4(10)
101 -> 5(10)
110 -> 6(10)
111 -> 7(10)
count = 8, min = 0, max = 7
```

* Формула:

```
count = 2^N, min = 0, max = 2^N - 1
```

* 1 байт (8 бит) - count = 256, min = 0, max = 255

## Отрицательные числа

29 - 5 -> 29 + (-5) = ?

1. Пусть для числа 5 мы используем 1 байт, тогда он будет записан как:

```
5 = 00000101
```

2. Получим обратный код для числа 5

```
00000101 -> 11111010
```

3. Получим дополнительный код

```
11111010 + 1 = 11111011
```
