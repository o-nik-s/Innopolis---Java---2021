## Фреймворк SCRUM + Kanban

* Backlog - содержит User Stories

* Спринт - от 2-х недель до 2-х месяцев, набор задач, который приводит к инкременту проекта.

* Планирование спринта - для каждой задачи определяем трудозатраты.

* Ежедневные встречи

* Ретроспектива
